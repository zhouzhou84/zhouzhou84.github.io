###插入代码（一般用不上，学计算机的还是放上来表示尊重）

如果是段落上的一个函数或片段的代码可以用反引号把它包起来（`），例如：

printf() 函数

代码块：

代码区块使用 4 个空格或者一个制表符（Tab 键），这里每一行前面都需要有4 个空格或者一个制表符。例如：

#include<iostream>
using namespace std;
int main(){
    for(;;){
        cout<<"憨憨慧，I love you❤️"<<endl;
    }
}

你也可以用 ` 包裹一段代码，并指定一种语言（也可以不指定）：

$(document).ready(function () {

####转义
Markdown 使用了很多特殊符号来表示特定的意义，如果需要显示特定的符号则需要使用转义字符，Markdown 使用反斜杠转义特殊字符：

文本加粗

** 我是星号 **

#我是井号#

Markdown 支持以下这些符号前面加上反斜杠来帮助插入普通的符号：
\ \ 反斜线

` 反引号

* 星号

_ 下划线

{} 花括号

[] 方括号

() 小括号

# 井号

+ 加号

- 减号

. 英文句号

! 感叹号
